Tadeáš Souček

1: NodeJS je runtime, který spouští JavaScript pomocí V8 enginu mimo prohlížeč. Navíc přidá do JS interface pro import knihoven: require, který může načítat i knihovny, který umí interagovat se systémem jako normální desktop aplikace. Lze použít pro vývoj backendu webu

2: V Browseru, nebo pomocí Node.JS (příp kdekoli, kde je interpreter (např. extensions do programů))

3: NPM instaluje knihovny spustitelné přes Node.JS (nebo bootstrap spouštěný na klientovi).

4: Instalace:

4.1: Linux: Stažení balíčku z repo pomocí systémového package manageru. Windows: Jít na npm.com, stáhnout exe soubor, ten spustit a proklikat

5: Informace o balíčku, name, dependencies, verze, author

6: Bootstrap je již předudělaný CSS, JS
